AEDs 1 - Prática

Descrição

Aplicação desenvolvida para a matéria de Algoritmos e Estruturas de Dados 1. O projeto consiste em um sistema de gerenciamento para uma base de dados, permitindo operações como inserção, remoção, busca e a geração de estatísticas sobre os dados armazenados. O objetivo principal foi praticar os conceitos de estruturas de dados dinâmicas e manipulação de arquivos em C++.
